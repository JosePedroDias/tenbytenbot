# tenbytenbot

trying to do a bot to play the [tenbyten](https://github.com/JosePedroDias/tenbyten) game via its HTTP interface

**work in progress**



## Installation

fetch the standalone jar from the `dist` folder



## Usage

    $ java -jar tenbytenbot-0.1.0-standalone.jar
